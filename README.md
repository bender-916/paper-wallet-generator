# Paper Wallet Generator

Generador offline de paper wallets para criptomonedas.

## Características
- Generación de wallets BTC/ETH sin conexión a internet
- QR codes para direcciones y claves privadas
- Compatible con BIP39 (mnemonics)
- 100% offline por seguridad

## Uso
```bash
python generate.py --coin btc --output wallet.pdf
```
